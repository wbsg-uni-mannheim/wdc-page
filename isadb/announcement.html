Web Data Commons - WebIsADb

Web Data Commons and Data and Web Science group University of Mannheim are pleased to announce that WebIsADb is offered as a publicly resource to the research community. WebIsADb is a database containing more than 400 million hypernymy relations we extracted from the CommonCrawl web corpus. This collection of relations represents a rich source of knowledge and may be useful for many researchers. We offer the tuple dataset for public download and an application programming interface to help other researchers programmatically query the database. The resource and more can be found here.
